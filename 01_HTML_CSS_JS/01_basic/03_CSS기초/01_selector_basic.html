CSS 선택자 기본

CSS

- HTML 문서의 스타일을 정의하는 스타일 시트 언어
- 캐스케이딩, 상속, 재사용성

1. 외부 스타일 시트 (.css)

- 별도의 .css 파일을 생성해 HTML 문서와 연결
Style.css
2. 내부 스타일 시트

- HTML 문서 내 head 안에 style을 사용하여 css 정의
<style> </style>
3. 인라인 스타일

- HTML 요소의 style 속성에 직접 스타일 적용
<div style="display: flex;"> </div>
